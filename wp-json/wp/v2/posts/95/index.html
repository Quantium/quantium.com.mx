Batalla entre el nokia N95 y el iPhone<\/p>\n

Apple a menudo ha mirado el iPhone como un tel\u00e9fono inteligente – a\u00fan as\u00ed tiene muchas competencias. Uno de los contendientes m\u00e1s fuertes es el N95 de Nokia<\/strong>. \u00bfCu\u00e1l es mejor? En realidad, la elecci\u00f3n depende de ti, pero esta comparaci\u00f3n puede ayudarte a tomar una decisi\u00f3n mucho m\u00e1s r\u00e1pida y f\u00e1cil:<\/p>\n

He aqui porqu\u00e9 el N95 puede ser el mejor:<\/strong><\/p>\n

Una de las mejores ventajas es la red GPS, que te permite navegar m\u00e1s f\u00e1cilmente. El desarrollo de este programa es el secreto para encontrar direcciones, ubicaciones, los nombres de las calles, e incluso los c\u00f3digos postales. Por un costo adicional, puedes actualizar este servicio para usarlo por medio de comandos de voz a trav\u00e9s de un equipo manos libres. Tambi\u00e9n aguanta bien los programas para ver tu email y puedes sincronizarlos con Microsoft Exchange Server. Esto permite que veas cualquier documento de la oficina a trav\u00e9s de un programa llamado QuickOffice. Tambi\u00e9n optimizar\u00e1 estos documentos para que puedas leerlos eficazmente. Esto es una de las caracter\u00edsticas m\u00e1s importantes usadas por usuarios de negocios. A diferencia del iPhone, est\u00e1 equipado con un radio de FM interno. Los auriculares te permiten acceder al radio y tambi\u00e9n pueden funcionar como sintonizador.<\/p>\n

Porqu\u00e9 el iPhone debe ser tu elecci\u00f3n:<\/strong><\/p>\n

Primero que nada, el iPhone no es solo un tel\u00e9fono. Tambi\u00e9n es un iPod. Puedes o\u00edr y ver todo tipo de archivos siempre y cuando este en formatos aceptados. Puedes o\u00edr m\u00fasica, ver una pel\u00edcula o un programa, o escuchar audio libros. Su pantalla de 3.5 pulgadas maximiza la vista para que tengas un entretenimiento completo. El tel\u00e9fono tiene capacidad para aceptar emails en forma de p\u00e1gina y en tiempo real, siempre y cuando no sea en formato HTML. Puedes configurar tu yahoo! Correo para eso.<\/p>\n

El iPhone hace uso de un explorador de red llamado Safari, que es muy avanzado. Puedes ver las p\u00e1ginas de la red tal cual son. Y puedes aumentarlas con solo tocar la pantalla.<\/p>\n

Apple quiere asegurarse de que tengas el mini OS adecuado para tu tel\u00e9fono. Esto significa que puedas disfrutar de aplicaciones de escritorio, al igual que las funciones b\u00e1sicas del tel\u00e9fono. Calendario, una libreta para notas, y una libreta de direcciones, entre otros. Puedes hacer varias tareas mientras navegas en la red y descargas tus emails, permiti\u00e9ndote ahorrar mucho tiempo.<\/p>\n

Tal vez es la interfaz de usuario del tel\u00e9fono por la que el iPhone es famoso. Integra la tecnolog\u00eda de pantalla digital, esto quiere decir que no tienes que usar ning\u00fan teclado s\u00f3lo enviar el texto o hacer la llamada. Tambi\u00e9n usa el teclado virtual QWERTY l, que aparece solamente cuando es usado.<\/p>\n

Cual dir\u00edas que es mejor? Aun es el punto de vista del usuario el que m\u00e1s importa.<\/p>\n
