Batalla entre Nokia N95 y el iPhone

by Andrés González Aragón|Published 24/08/2007|1 comment

Por: El Señor de los Gadgets

Batalla entre el nokia N95 y el iPhone

Apple a menudo ha mirado el iPhone como un teléfono inteligente – aún así tiene muchas competencias. Uno de los contendientes más fuertes es el N95 de Nokia. ¿Cuál es mejor? En realidad, la elección depende de ti, pero esta comparación puede ayudarte a tomar una decisión mucho más rápida y fácil:

He aqui porqué el N95 puede ser el mejor:

Una de las mejores ventajas es la red GPS, que te permite navegar más fácilmente. El desarrollo de este programa es el secreto para encontrar direcciones, ubicaciones, los nombres de las calles, e incluso los códigos postales. Por un costo adicional, puedes actualizar este servicio para usarlo por medio de comandos de voz a través de un equipo manos libres. También aguanta bien los programas para ver tu email y puedes sincronizarlos con Microsoft Exchange Server. Esto permite que veas cualquier documento de la oficina a través de un programa llamado QuickOffice. También optimizará estos documentos para que puedas leerlos eficazmente. Esto es una de las características más importantes usadas por usuarios de negocios. A diferencia del iPhone, está equipado con un radio de FM interno. Los auriculares te permiten acceder al radio y también pueden funcionar como sintonizador.

Porqué el iPhone debe ser tu elección:

Primero que nada, el iPhone no es solo un teléfono. También es un iPod. Puedes oír y ver todo tipo de archivos siempre y cuando este en formatos aceptados. Puedes oír música, ver una película o un programa, o escuchar audio libros. Su pantalla de 3.5 pulgadas maximiza la vista para que tengas un entretenimiento completo. El teléfono tiene capacidad para aceptar emails en forma de página y en tiempo real, siempre y cuando no sea en formato HTML. Puedes configurar tu yahoo! Correo para eso.

El iPhone hace uso de un explorador de red llamado Safari, que es muy avanzado. Puedes ver las páginas de la red tal cual son. Y puedes aumentarlas con solo tocar la pantalla.

Apple quiere asegurarse de que tengas el mini OS adecuado para tu teléfono. Esto significa que puedas disfrutar de aplicaciones de escritorio, al igual que las funciones básicas del teléfono. Calendario, una libreta para notas, y una libreta de direcciones, entre otros. Puedes hacer varias tareas mientras navegas en la red y descargas tus emails, permitiéndote ahorrar mucho tiempo.

Tal vez es la interfaz de usuario del teléfono por la que el iPhone es famoso. Integra la tecnología de pantalla digital, esto quiere decir que no tienes que usar ningún teclado sólo enviar el texto o hacer la llamada. También usa el teclado virtual QWERTY l, que aparece solamente cuando es usado.

Cual dirías que es mejor? Aun es el punto de vista del usuario el que más importa.
